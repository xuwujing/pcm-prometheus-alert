# 其他智能体接手说明

## 当前状态

项目已经完成第一阶段骨架：

1. Maven 父工程已建立
2. `core` / `starter` / `demo` 已可编译
3. demo 已验证异常告警、慢请求告警、mock webhook
4. `mvn test` 通过

## 先看哪些文件

优先阅读：

1. `README.md`
2. `docs/10-详细设计说明.md`
3. `docs/08-MVP任务拆分.md`
4. `pcm-prometheus-alert-spring-boot-starter/src/main/java/com/pcm/alert/starter/PcmAlertAutoConfiguration.java`
5. `pcm-prometheus-alert-core/src/main/java/com/pcm/alert/core/AlertManager.java`

## 下一批建议任务

建议从以下任务里任选一个开始，不要同时大改多个方向：

1. 完善异常告警：异常过滤、堆栈裁剪、traceId 提取策略
2. 完善请求告警：路径排除、状态码维度、请求体摘要
3. 完善指标告警：CPU 采样、恢复事件、规则化阈值
4. 完善测试：starter 自动装配测试、webhook publisher 测试
5. 完善文档：README 接入示例、配置说明、发布说明
